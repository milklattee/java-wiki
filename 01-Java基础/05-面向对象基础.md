# 面向对象基础 (OOP)

## 三大特性

1. **封装** — 隐藏内部实现，暴露 public 接口
2. **继承** — 子类复用父类代码，`extends` 关键字
3. **多态** — 同一接口，多种实现形式

## 类与对象

```java
public class Person {
     private String name;   // 字段
     private int age;

     public Person(String name, int age) {  // 构造方法
         this.name = name;
         this.age = age;
     }

     public void sayHello() {
         System.out.println("Hello, I'm " + name);
     }

     // Getter / Setter
     public String getName() { return name; }
     public void setName(String name) { this.name = name; }
}

Person p = new Person("Alice", 25);
p.sayHello();
```

## 关键概念

- **构造方法** — 名称与类名相同、无返回值；未定义时编译器提供默认无参构造
- **this** — 当前对象引用；`this()` 调用本类构造器
- **static** — 类级别成员，不属于实例；静态方法不能直接访问实例成员
- **final** — 修饰类不可继承、方法不可重写、变量不可修改
