# 控制流程

## if-else

```java
if (条件) {
     // 条件为 true 执行
} else if (另一个条件) {
     // ...
} else {
     // 都不满足时执行
}
```

## switch

**JDK 14+** 支持箭头语法和 `yield` 返回值：

```java
String result = switch (day) {
     case 1, 2, 3, 4, 5 -> "工作日";
     case 6, 7 -> "周末";
     default -> "无效";
};
```

## 循环

### for

```java
for (int i = 0; i < 10; i++) System.out.println(i);

// 增强 for (for-each)
for (String s : list) System.out.println(s);
```

### while

```java
while (条件) { /* 先判断再执行 */ }
do { /* 至少执行一次 */ } while (条件);
```

## break / continue / 标签

```java
outer:
for (int i = 0; i < 3; i++) {
     for (int j = 0; j < 3; j++) {
         if (i * j > 2) break outer;
     }
}
```
