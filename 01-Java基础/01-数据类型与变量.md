# 数据类型与变量

## 基本数据类型（8种）

| 类型 | 大小 | 默认值 | 范围 |
|------|------|--------|------|
| `byte` | 1字节 | 0 | -128 ~ 127 |
| `short` | 2字节 | 0 | -32768 ~ 32767 |
| `int` | 4字节 | 0 | -2^31 ~ 2^31-1 |
| `long` | 8字节 | 0L | -2^63 ~ 2^63-1 |
| `float` | 4字节 | 0.0f | IEEE 754 单精度 |
| `double` | 8字节 | 0.0d | IEEE 754 双精度 |
| `char` | 2字节 | '\u0000' | 0 ~ 65535 (Unicode) |
| `boolean` | JVM决定 | false | true / false |

## 引用数据类型

- **类 (class)** — 用户自定义类型
- **接口 (interface)** — 行为契约
- **数组 (array)** — 同类型元素的集合
- **枚举 (enum)** — 固定的常量集合
- **String** — 不可变字符序列（特殊引用类型）

## 变量声明

```java
// 局部变量 — 必须初始化后才能使用
int count = 10;

// 实例变量 — 有默认值
class Example {
     String name;  // null
     int age;      // 0
}

// 静态变量（类变量）
static final double PI = 3.14159;
```

## 类型转换

- **自动类型转换**（隐式）: `byte → short → int → long → float → double`
- **强制类型转换**（显式）: `(目标类型) 值`，可能丢失精度

```java
int i = 100;
long l = i;           // 自动转换 OK
double d = 3.14;
int j = (int) d;      // 强制转换，j = 3（丢失小数）
```

## 包装类

每种基本类型都有对应的包装类：`Integer`, `Long`, `Double` 等。JDK 5+ 支持自动装箱/拆箱。

```java
Integer num = 42;       // 自动装箱 int → Integer
int val = num;          // 自动拆箱 Integer → int
```
