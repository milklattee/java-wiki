# 数组

## 声明与初始化

```java
int[] arr = new int[5];          // 动态初始化
int[] arr2 = {1, 2, 3, 4, 5};   // 静态初始化
int[] arr3;
arr3 = new int[]{1, 2, 3};
```

## 关键属性

- `arr.length` — 长度（属性，不是方法）
- 数组是**定长**的，创建后长度不可变
- 索引从 `0` 开始，越界抛 `ArrayIndexOutOfBoundsException`

## 遍历方式

```java
// 传统 for
for (int i = 0; i < arr.length; i++) { /* arr[i] */ }

// for-each
for (int num : arr) { /* num */ }

// Stream (JDK 8+)
Arrays.stream(arr).forEach(System.out::println);
```

## 多维数组

```java
int[][] matrix = new int[3][4];   // 3行4列
int[][] jagged = new int[3][];    // 不规则数组
jagged[0] = new int[2];
jagged[1] = new int[5];
```

## Arrays 工具类常用方法

| 方法 | 说明 |
|------|------|
| `Arrays.sort(arr)` | 排序 |
| `Arrays.binarySearch(arr, key)` | 二分查找（需先排序）|
| `Arrays.fill(arr, val)` | 填充 |
| `Arrays.copyOf(arr, n)` | 复制并调整长度 |
| `Arrays.toString(arr)` | 转字符串 |
| `Arrays.equals(a, b)` | 比较内容 |
