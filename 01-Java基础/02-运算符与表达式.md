# 运算符与表达式

## 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 / 字符串拼接 | `a + b`, `"Hello" + "World"` |
| `-` | 减法 | `a - b` |
| `*` | 乘法 | `a * b` |
| `/` | 除法（整数除得整数） | `5 / 2 → 2` |
| `%` | 取模（余数） | `5 % 2 → 1` |
| `++` | 自增 | `a++`, `++a` |
| `--` | 自减 | `a--`, `--a` |

## 关系运算符

`==`, `!=`, `>`, `<`, `>=`, `<=` — 返回 `boolean` 类型。

> ⚠️ 引用类型用 `==` 比较的是内存地址，内容比较用 `.equals()`。

## 逻辑运算符

| 运算符 | 说明 | 短路 |
|--------|------|------|
| `&&` | 逻辑与 | 是（左侧 false 则不执行右侧） |
| `\|\|` | 逻辑或 | 是（左侧 true 则不执行右侧） |
| `!` | 逻辑非 | — |
| `&` | 按位与 / 非短路与 | 否 |
| `\|` | 按位或 / 非短路或 | 否 |

## 位运算符

```java
int a = 0b1100;  // 12
int b = 0b1010;  // 10
a & b;   // 0b1000 = 8  按位与
a | b;   // 0b1110 = 14 按位或
a ^ b;   // 0b0110 = 6  按位异或
~a;      // 按位取反
a << 1;  // 左移 = 乘以2
a >> 1;  // 右移 = 除以2（带符号）
a >>> 1; // 无符号右移
```

## 三元运算符

```java
int max = (a > b) ? a : b;
```
