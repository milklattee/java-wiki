# JVM 虚拟机

## 运行时数据区

| 区域 | 线程 | 内容 | 异常 |
|------|------|------|------|
| 程序计数器 | 私有 | 当前字节码行号 | — |
| 虚拟机栈 | 私有 | 栈帧（局部变量、操作数栈）| StackOverflowError |
| 本地方法栈 | 私有 | Native 方法 | StackOverflowError |
| **堆** | 共享 | 对象实例、数组 | OutOfMemoryError |
| **方法区** | 共享 | 类信息、常量、静态变量 | OutOfMemoryError |

## 类加载过程

```
加载 → 验证 → 准备 → 解析 → 初始化 → 使用 → 卸载
```

## 类加载器

- **Bootstrap ClassLoader** — 加载核心类库 (`rt.jar`)
- **Extension ClassLoader** — 扩展目录
- **Application ClassLoader** — 应用 classpath
- **双亲委派模型** — 先让父加载器加载，避免重复加载核心类

## 垃圾回收 (GC)

### 判断对象死亡

1. **引用计数法** — 有循环引用问题
2. **可达性分析** — GC Roots 不可达则回收

### 垃圾回收算法

| 算法 | 说明 |
|------|------|
| 标记-清除 | 标记垃圾 → 清除，产生内存碎片 |
| 标记-整理 | 标记 → 存活对象前移 → 清理边界外 |
| 复制 | 分两块区域，存活对象复制到另一块 |
| 分代收集 | 新生代用复制，老年代用标记-整理 |

### 常见收集器

| 收集器 | 区域 | 特点 |
|--------|------|------|
| Serial | 新生代 | 单线程，适合客户端 |
| Parallel Scavenge | 新生代 | 吞吐量优先 |
| CMS | 老年代 | 低停顿，并发标记 |
| **G1** | 全部 | 可预测停顿，JDK 9+ 默认 |
| **ZGC** | 全部 | 超低延迟 <1ms (JDK 15+) |

## JVM 参数

```bash
-Xms512m          # 初始堆大小
-Xmx2g            # 最大堆大小
-Xss256k          # 线程栈大小
-XX:+UseG1GC      # 使用 G1 收集器
-XX:MaxGCPauseMillis=200 # G1 最大停顿目标
```
