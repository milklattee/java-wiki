# 多线程与并发

## 创建线程

```java
// 方式1：继承 Thread
class MyThread extends Thread {
     public void run() { System.out.println("线程运行"); }
}
new MyThread().start();

// 方式2：实现 Runnable（推荐）
new Thread(() -> System.out.println("Lambda 线程")).start();

// 方式3：Callable + Future（有返回值）
ExecutorService executor = Executors.newFixedThreadPool(4);
Future<String> future = executor.submit(() -> "result");
String result = future.get();  // 阻塞等待
executor.shutdown();
```

## 线程安全

### synchronized

```java
// 同步方法
public synchronized void increment() { count++; }

// 同步块
synchronized (lock) { count++; }
```

### Lock (java.util.concurrent.locks)

```java
ReentrantLock lock = new ReentrantLock();
lock.lock();
try { count++; }
finally { lock.unlock(); }
```

## 线程通信

- `wait()` / `notify()` / `notifyAll()` — Object 方法，必须在 synchronized 中调用
- `CountDownLatch` — 等待一组操作完成
- `CyclicBarrier` — 多个线程互相等待
- `Semaphore` — 控制同时访问资源的线程数

## 线程池

```java
// 固定大小线程池
ExecutorService pool = Executors.newFixedThreadPool(10);
pool.execute(() -> doWork());

// 推荐使用 ThreadPoolExecutor 显式配置
ThreadPoolExecutor tpe = new ThreadPoolExecutor(
     4, 8, 60L, TimeUnit.SECONDS,
     new LinkedBlockingQueue<>(100),
     new ThreadPoolExecutor.CallerRunsPolicy()
);
```

## 并发集合

| 集合 | 说明 |
|------|------|
| `ConcurrentHashMap` | 线程安全 HashMap |
| `CopyOnWriteArrayList` | 写时复制，读多写少场景 |
| `BlockingQueue` | 生产者-消费者模式 |
