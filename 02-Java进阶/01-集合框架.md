# Java 集合框架

## 架构概览

```
Collection                 Map
├── List                   ├── HashMap
│   ├── ArrayList          ├── LinkedHashMap
│   └── LinkedList         ├── TreeMap
├── Set                    └── ConcurrentHashMap
│   ├── HashSet
│   ├── LinkedHashSet
│   └── TreeSet
└── Queue / Deque
     ├── PriorityQueue
     └── ArrayDeque
```

## List — 有序可重复

| 实现 | 底层 | 特点 |
|------|------|------|
| `ArrayList` | Object[] | 随机访问 O(1)，尾部增删快 |
| `LinkedList` | 双向链表 | 头尾增删 O(1)，随机访问 O(n) |

```java
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.get(0);               // 索引访问
list.remove("Python");
```

## Set — 无序不重复

| 实现 | 特点 |
|------|------|
| `HashSet` | 基于 HashMap，O(1) 增删查 |
| `LinkedHashSet` | 维护插入顺序 |
| `TreeSet` | 红黑树，元素有序 O(log n) |

## Map — 键值对

| 实现 | 特点 |
|------|------|
| `HashMap` | O(1) 存取，允许 null key |
| `LinkedHashMap` | 维护插入/访问顺序 |
| `TreeMap` | 红黑树，key 有序 |
| `ConcurrentHashMap` | 线程安全，分段锁 |

```java
Map<String, Integer> map = new HashMap<>();
map.put("apple", 3);
map.get("apple");          // 3
map.getOrDefault("banana", 0);  // 0

// 遍历
for (Map.Entry<String, Integer> e : map.entrySet()) {
     System.out.println(e.getKey() + "=" + e.getValue());
}
```

## Collections 工具类

```java
Collections.sort(list);
Collections.reverse(list);
Collections.shuffle(list);
Collections.max(list);
Collections.synchronizedList(list);  // 获取线程安全包装
```
