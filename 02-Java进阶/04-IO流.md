# IO 流

## 分类

| 维度 | 分类 |
|------|------|
| 流向 | 输入流 `InputStream` / `Reader`，输出流 `OutputStream` / `Writer` |
| 数据单位 | 字节流 (Stream, 8bit) / 字符流 (Reader/Writer, 16bit) |
| 角色 | 节点流 (直接读写) / 处理流 (装饰/包装) |

## 字节流

```java
// 文件复制
try (FileInputStream fis = new FileInputStream("src.bin");
      FileOutputStream fos = new FileOutputStream("dst.bin")) {
     byte[] buf = new byte[8192];
     int n;
     while ((n = fis.read(buf)) != -1) {
         fos.write(buf, 0, n);
     }
}
```

## 字符流

```java
try (BufferedReader br = new BufferedReader(new FileReader("input.txt"));
      BufferedWriter bw = new BufferedWriter(new FileWriter("output.txt"))) {
     String line;
     while ((line = br.readLine()) != null) {
         bw.write(line);
         bw.newLine();
     }
}
```

## NIO (JDK 1.4+)

- `Path` / `Paths` / `Files` — 现代文件操作 API
- `Channel` + `Buffer` — 非阻塞 IO 基础

```java
Path path = Path.of("data.txt");
List<String> lines = Files.readAllLines(path);
Files.writeString(Path.of("out.txt"), "Hello NIO");

// 遍历目录
try (Stream<Path> walk = Files.walk(Path.of("src"))) {
     walk.filter(Files::isRegularFile).forEach(System.out::println);
}
```
